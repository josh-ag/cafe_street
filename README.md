## Getting Started

Make sure Nodejs is installed  in the development machine

```bash
$git clone this_repo
$cd project_directory

NPM:
$npm install
#Then run the following command to spin-up the dev-server
$npm run dev

#OR

YARN:
$yarn install
#Then run the following command to spin-up the dev-server
$yarn run dev






```
